\# RNN Stock Price Prediction



This project analyzes historical stock market data for AMZN, GOOGL, IBM and MSFT and develops Recurrent Neural Network models for stock price prediction.



\## Models Implemented

\- Simple RNN

\- LSTM



\## Dataset

Historical stock market data from 2006–2018.



\## Key Findings

\- A 30-day window size was selected.

\- Simple RNN achieved better test performance than LSTM on this dataset.

\- Both models captured overall trends but struggled with large price increases in later years.
